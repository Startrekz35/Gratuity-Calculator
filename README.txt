Tip Calculator
Description
The Tip Calculator is a simple Python program that helps users calculate the total bill including a tip and divides the amount equally among a group of people. The user inputs the total bill, selects a tip percentage, and specifies the number of people sharing the bill, and the program outputs how much each person should pay.

Features
Takes the total bill as input.
Calculates the tip based on a user-specified percentage.
Divides the total amount among a group of people.
Rounds the result to two decimal places.
Requirements
To run this project, you need:

Python 3.x installed on your system.

How to Run the Program
1. Clone or download the repository to your local machine.
2. Open a terminal or command prompt in the directory containing the script.
3. Run the following command:
    python Tip_Calculator.py
4. The program will prompt you to enter:
        The total bill amount.
        The percentage tip you want to give.
        The number of people splitting the bill.
        The result will display how much each person should pay.

Example:
    Welcome to the tip calculator!
    What was the total bill? $100
    What percentage tip would you like to give? 15
    How many people to split the bill? 4
    Each person should pay: $28.75
